# ПомРеж 0.0.8 — инструкция для тестирования (macOS)

## Установка приложения

1. Откройте папку `dist/`.
2. Запустите файл `ПомРеж-0.0.8.dmg`.
3. Перетащите приложение `ПомРеж` в папку `Applications` (Программы).

## Первый запуск (Gatekeeper)

Если macOS блокирует запуск неподписанного приложения:

1. Найдите `ПомРеж` в `Applications`.
2. Сделайте **Control + Click** по приложению.
3. Выберите `Открыть`.
4. Подтвердите запуск в системном диалоге.

## Что проверить в приложении

- Открытие/сохранение шоу-файлов через нативные диалоги.
- Открытие экранов `Stage`, `Tech`, `Speaker`.
- Корректная работа таймера: старт, пауза/продолжить, DONE.
- Меню `OSC` и тест OSC-соединения.
- Смена тем (тёмная/светлая/высокий контраст).
- Модалка настройки экрана ведущего.

## Горячие клавиши

- `Space` — следующий cue.
- `P` — пауза/продолжить таймер.
- `Cmd+S` / `Ctrl+S` — сохранить.
- `Cmd+Shift+S` / `Ctrl+Shift+S` — сохранить как.
- `Cmd+O` / `Ctrl+O` — открыть шоу.
- `Cmd+1` / `Ctrl+1` — Stage.
- `Cmd+2` / `Ctrl+2` — Tech.
- `Cmd+3` / `Ctrl+3` — Speaker.

## OSC для теста

- В десктоп-версии встроенный OSC HTTP/UDP сервер запускается автоматически при старте приложения.
- Для локальной проверки используйте утилиту:
  - `node test_osc_receiver.js`
- Затем в приложении выполните `OSC -> Тест соединения`.

## Что приложить к фидбеку

- Версию macOS.
- Что именно тестировали (меню/таймер/экраны/OSC).
- Шаги воспроизведения проблемы.
- Скриншот или короткое видео при наличии визуального бага.
